# 🎨 Abhyush Dixit Portfolio

A modern, high-performance portfolio website built with React, Vite, and Tailwind CSS.

## 🚀 Tech Stack

- React (TypeScript)
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## 📁 Features

- Smooth animations
- Responsive design
- Light/Dark mode
- Project gallery with lightbox
- Resume download

## 🛠️ Installation

```bash
npm install
npm run dev
